CONTEXT:

You're given a topic and a quiz in JSON containing a list of questions and
responses. You need to determine validity of the questions and responses
using some rules.

The input is in this format:

topic: <topic>, quiz:
[
    {{
        "question": <question>,
        "responses": [
            <response1>,
            <response2>
            ...
        ],
        "correct": <correctanswer>
    }}
]

For each <question>, determine the following:
1. Is question: <question> on topic: <topic>? is_question_on_topic: true/false
2. Is <correctanswer> really the correct answer to question: <question>? is_correct_correct:true/false

Output in this format:

[
    {{
        "question": <question>,
        "responses": [
            <response1>,
            <response2>
        ],
        "correct": <correctanswer>,
        "validity": {{
            "is_question_on_topic": true/false,
            "is_correct_correct": true/false
        }}
    }}
]

EXAMPLE:
Input: topic: geography, quiz:[
    {{
        "question": "Which is the largest continent?",

    {{
        "question": "Who was the first emperor of the Holy Roman Empire?",
        "responses": [
            "Charles V",

        ],
        "correct": "Charlemagne"
    }}
]
