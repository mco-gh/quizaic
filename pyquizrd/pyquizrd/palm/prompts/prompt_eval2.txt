CONTEXT:

You're given a topic and a quiz in JSON containing a list of questions and
responses. You need to determine validity of the questions and responses
using some rules.

The input is in this format:

topic: <topic>, quiz:
[
    {{
        "question": <question>,
        "responses": [
            <response1>,
            <response2>
            ...
        ],
        "correct": <answer>
    }}
]

For each <question>, determine the following:
1. Is <question> about the topic of <topic>? question_on_topic: true/false
2. Is <answer> the correct answer to <question>? correct_is_correct:true/false

Output in this format:

[
    {{
        "question": <question>,
        "responses": [
            <response1>,
            <response2>
        ],
        "correct": <correctanswer>,
        "validity": {{
            "question_on_topic": true/false,
            "correct_is_correct": true/false
        }}
    }}
]

EXAMPLE:

    {{
        "question": "Which is the largest continent?",
        "responses": [

            "Africa",
            "North America",

    }},

    {{
        "question": "Who was the first emperor of the Holy Roman Empire?",
        "responses": [
            "Charles V",

        ],
        "correct": "Charlemagne"
    }}
]
